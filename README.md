## About
A website for discovering and searching for films for a village cinema.

**Link to the repository :** [https://github.com/Physicprog/1WEBD-Griffon--Nicolas-Khylian.git](https://github.com/Physicprog/1WEBD-Griffon--Nicolas-Khylian.git)

## The pages

### Home
Discover the trending movies of the moment with their poster, title and a short synopsis.

### Search
Search for your favorite movies in real time and explore the results.

### Detail Page
View all the information about a film: synopsis, genre, actors, ratings and release date.

- Display of trending movies
- Infinite scrolling to display all movies seamlessly
- Real-time search without reloading
- Detailed information on each movie
- Mobile and desktop optimized design
- Accessible and intuitive interface
